# 🚀 3D Responsive Portfolio Website

A stunning, fully responsive, and highly animated **Single Page Application (SPA)** built to showcase my software development expertise, projects, and skills. This portfolio features immersive 3D graphics, seamless fluid animations, and a direct contact system.

---

## ✨ Features

- **🌐 Single Page Application:** Fast, seamless user experience without page reloads using React.js.
- **🎨 Immersive 3D Visuals:** Interactive 3D models and backgrounds powered by **Three.js** and **React Three Fiber**.
- **🎬 Smooth Animations:** Eye-catching transitions, scroll-triggered animations, and micro-interactions built with **Framer Motion**.
- **📱 Fully Responsive Design:** Modern, clean, and pixel-perfect UI crafted using **Tailwind CSS**.
- **📬 Functional Contact Form:** Direct email communication system integrated via **Email.js** (No backend server required!).

---

## 🛠️ Tech Stack

- **Core Library:** React.js (Vite)
- **3D Graphics:** Three.js & `@react-three/fiber` & `@react-three/drei`
- **Animation:** Framer Motion
- **Styling:** Tailwind CSS
- **Contact Service:** Email.js
